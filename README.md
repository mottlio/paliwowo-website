# paliwowo landing page

Simple bilingual (PL/EN) landing page for the paliwowo app.

## Behavior
- Defaults to **Polish** for Polish IPs (via `ipapi.co` country detection).
- Falls back to browser language (`pl` -> Polish, otherwise English).
- Users can manually switch language with the `PL/EN` buttons.

## Local preview
Open `index.html` directly or serve with any static server.
